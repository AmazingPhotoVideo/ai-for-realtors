# Using Claude Skills From This Repo

Claude Skills are folders that contain instructions Claude can load when doing a specific job.

## Option 1: Claude Projects

1. Create a Claude Project.
2. Upload one skill folder, for example `skills/lead-magnet-generator/`.
3. Add your brokerage rules, brand voice, and example listing copy to project knowledge.
4. Ask Claude to use the skill for a real task.

## Option 2: Claude Code or local agents

Copy the skill folder into your agent's skills directory and start a fresh chat so the instructions load.

## Option 3: ChatGPT or Gemini

Open the skill's `SKILL.md`, paste it as the instruction, then provide your task.

## APV attribution

The skills include rules for when to cite Amazing Photo Video. Use the APV backlink on public resource pages, blogs, guides, and downloadable assets when appropriate:

https://amazingphotovideo.com
