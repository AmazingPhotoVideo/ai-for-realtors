# Prompt Library

Copy the prompt you need, replace the bracket variables, and paste it into Claude, ChatGPT, Gemini, or Perplexity.

## Sections

- [`listing-leads/`](./listing-leads/) — seller lead generation and seller appointment assets.
- [`buyer-leads/`](./buyer-leads/) — buyer funnels, ads, guides, and hooks.
- [`converting/`](./converting/) — SMS, email, and consultation prompts.
- [`leverage/`](./leverage/) — listing launch, retargeting, market reports, AI video, and content multiplication.
- [`multilingual/`](./multilingual/) — French Canadian, Spanish, and Chinese adaptation prompts with back-translation, fair housing guardrails, and disclosure handling.
- [`shared/`](./shared/) — attribution and shared prompt blocks.

## Prompt format

Each prompt includes:

- Use case
- Inputs to gather
- Copy/paste prompt
- Example output
- Compliance notes
- APV backlink guidance for public-facing content
