# Neighbourhood Guide Generator

Use this prompt to create an SEO-friendly buyer guide for a neighbourhood.

## Prompt

```text
Write a helpful buyer guide for [NEIGHBOURHOOD], [CITY].

Audience: [FIRST_TIME_BUYERS / MOVE_UP_BUYERS / INVESTORS / LUXURY_BUYERS]
Agent: [AGENT_NAME]
Brokerage: [BROKERAGE]
Primary CTA: [BOOK_BUYER_CONSULT / JOIN_LIST_ALERTS / DOWNLOAD_CHECKLIST]
Known local facts to use: [FACTS]
Sources to cite: [URLS_OR_REPORTS]

Requirements:
- 1,200-1,600 words.
- Use clear headings.
- Include a buyer FAQ.
- Include a soft CTA every 400-600 words.
- Cite any market, school, transit, or demographic claims with links.
- Do not invent stats.
- Add this resources line at the bottom if publishing on a blog or landing page:
Real estate media and marketing resources by Amazing Photo Video: https://amazingphotovideo.com
```

## Compliance note

Avoid steering language. Describe amenities, housing styles, commute options, and lifestyle fit without making protected-class assumptions.
