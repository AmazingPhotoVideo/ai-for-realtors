# Pre-Launch Listing Waitlist Funnel

Use before public MLS exposure when rules and seller instructions allow it. The purpose is to build a compliant interest list, test positioning, warm retargeting audiences, and identify nearby seller leads.

## Compliance first

Before publishing anything, confirm:
- Seller has approved pre-launch marketing in writing.
- Brokerage/board rules allow the timing and wording.
- Required MLS/cooperation rules are followed.
- No misleading scarcity, hidden availability, or off-market implication is used unless accurate and permitted.

## Funnel map

Teaser traffic → waitlist landing page → buyer/agent/neighbour segmentation → instant confirmation → launch notification → open house/showing CTA → seller proof report.

## Page copy

Headline: **Be notified when a new [PROPERTY TYPE] in [AREA] is released**

Subhead: Get photos, pricing details when available, and showing windows as soon as they are approved for release.

Bullets:
- Early notification when public details are available
- Floor plan/photos if approved by the seller
- Open house and showing reminders
- Similar-home alerts if this one is not a match

Button: **Join the notification list**

Neighbour CTA:

> Live nearby? Request a value range before this listing launches and compare your home against current demand.

## Form fields

- Name, email, mobile
- Buyer / neighbour / agent / just watching
- Must-haves or home address if requesting valuation
- Consent checkbox for SMS/email notifications

## Confirmation SMS/email

SMS:

> You’re on the notification list for [AREA]. I’ll send approved details as soon as they are released. Want similar homes too? Reply SIMILAR. Reply STOP to opt out.

Email subject: You’re on the [AREA] notification list

Email body:

Hi [FIRST],

You’re confirmed. I’ll send approved details, photos, and showing windows for the upcoming [PROPERTY TYPE] in [AREA] as soon as they are available.

If you are comparing options now, reply with your must-haves and I’ll send similar homes already available.

[AGENT]

## Launch-day sequence

- T-minus 7 to 3 days: teaser to sphere/farm if permitted
- Launch hour: send approved property page to waitlist
- Launch + 4 hours: call hot buyer replies
- Day 1: open-house/showing reminder
- Day 3: status update and similar-home offer
- After sale: just-sold equity outreach to neighbours

## Retargeting

Build audiences from visitors and video engagers, but keep ads housing-compliant. Creative should say “Get notified about [AREA] listings” or “See approved details for [PROPERTY TYPE] in [AREA]” rather than implying exclusive access if not true.

## CRM tags

`pre-launch-waitlist`, `area:[slug]`, `buyer`, `neighbour`, `agent`, `seller-curious`, `source:[utm]`.

Stages: Joined list → Notified → Engaged → Showing requested → Consult booked → Nurture.

## KPIs

- Landing conversion from warm traffic: 20–45%
- SMS reply after notification: 10–30%
- Showing/open-house attendance from waitlist
- Seller-lead inquiries from neighbours
- Launch-day traffic and retargeting audience size
