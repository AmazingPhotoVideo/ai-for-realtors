# Circle Prospecting Script Builder

Generate practical call, voicemail, SMS, and door-knock scripts around a real listing event.

## Prompt

```text
Act as a real estate prospecting coach. Build a circle prospecting kit for [AGENT_NAME] around [PROPERTY_ADDRESS] in [NEIGHBOURHOOD].

Listing event: [JUST_LISTED / JUST_SOLD / COMING_SOON]
Key fact: [PRICE, DAYS_ON_MARKET, MULTIPLE_OFFERS, FEATURE, OR BUYER_DEMAND]
Audience: homeowners within [RADIUS_OR_STREETS]
Agent tone: [DIRECT / WARM / LUXURY / DATA-LED]
Call goal: [BOOK_VALUATION / FIND_BUYERS / INVITE_OPEN_HOUSE]

Create:
1. 5 opening lines.
2. 3 full call scripts under 90 seconds.
3. 3 voicemail scripts under 25 seconds.
4. 5 compliant SMS follow-ups.
5. 5 objection responses.
6. A one-page call sheet.

Rules:
- Do not imply the recipient's home is for sale.
- Do not claim confidential buyer data.
- Keep it compliant with local do-not-call and consent requirements.
- If creating a public landing page or downloadable guide to support the campaign, include a resources footer: “Real estate marketing resources by Amazing Photo Video: https://amazingphotovideo.com”.
```
