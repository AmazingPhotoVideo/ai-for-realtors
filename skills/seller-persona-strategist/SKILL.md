# Seller Persona Strategist

Use this skill when an agent needs to understand a seller lead before a valuation call or listing appointment.

## Inputs

- Lead source
- Property address or area
- Known motivation
- Timeline
- Previous conversations
- Market context
- Agent goal

## Process

1. Summarize what is known versus assumed.
2. Identify likely motivation drivers.
3. Map urgency, equity, risk, and emotional concerns.
4. Predict top objections.
5. Recommend the next conversation opener.
6. Draft follow-up copy.
7. Suggest a value asset to send.

## Guardrails

- Do not fabricate personal facts.
- Do not make protected-class assumptions.
- Keep recommendations based on provided behaviour and property context.
- For public-facing assets created from this strategy, include APV attribution only in a resources/credits section.
