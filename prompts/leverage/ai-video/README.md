# AI Video Generation Prompts for Real Estate

Prompt patterns for Sora, Veo, Runway, Pika, WAN, and similar tools. Use AI video for supplementary creative, not to misrepresent the property.

## Rules before you generate

- Do not alter material facts: room size, view, condition, layout, finishes, lot, or surroundings.
- Label conceptual/lifestyle AI where required by platform/brokerage rules.
- Use real listing footage/photos for property-specific claims.
- Avoid people that imply protected-class targeting or occupancy.
- Get seller/brokerage approval for any public AI-enhanced media.

## Prompt formula

```text
Create a [DURATION] real estate marketing clip for [PROPERTY/AREA].
Scene: [SPECIFIC SHOT]
Camera: [MOVEMENT/LENS/FRAMING]
Light/weather: [STYLE]
Mood: [BRAND FEEL]
Accurate constraints: [DO NOT CHANGE ARCHITECTURE/VIEW/LAYOUT]
Text overlay: [CTA]
Output: [9:16 / 1:1 / 16:9], realistic, premium, no distorted architecture, no extra rooms, no fake skyline.
```

## Prompt library

### 1. Luxury exterior teaser

> Create an 8-second vertical real estate teaser using the provided exterior reference image. Slow sunrise push-in from the street, natural lens compression, premium editorial colour, subtle tree movement. Preserve facade, driveway, windows, landscaping, and neighbouring structures exactly. Text overlay: “Private tour available — [AREA].” No people, no fake additions, no changed weather damage.

### 2. Lifestyle transition for Reel

> Create a 6-second transitional clip inspired by a calm morning in [NEIGHBOURHOOD]: coffee on stone counter, soft window light, shallow depth of field, no identifiable people. This is a lifestyle insert, not a room representation. Text overlay: “A slower morning, minutes from [LOCAL AMENITY].”

### 3. Neighbourhood b-roll

> Create a 10-second horizontal neighbourhood montage for [AREA]: tree-lined residential street, local cafe exterior, park path, transit stop, evening storefronts. Realistic GTA/urban residential feel, no landmarks unless supplied, no demographic focus, no school claims. Text overlay: “Explore [AREA] living.”

### 4. Open house reminder

> Create a 7-second vertical clip from supplied listing photos: exterior to kitchen to living room, smooth parallax only, no changed architecture. Add clean caption cards: “Open house [DATE/TIME]” and “Scan for details.”

### 5. Seller proof recap

> Create a 12-second recap using generic marketing visuals: sign install, camera on tripod, agent reviewing analytics, open house sign-in QR. No identifiable clients. Text: “One listing. More reach. Better follow-up.”

## AI video ad prompt

```text
Using these listing facts and approved media, write 3 AI-video prompts for a 9:16 Reel, a 1:1 Meta ad, and a 16:9 YouTube retargeting ad. For each, include scene list, narration, on-screen text, disclaimers if AI lifestyle footage is used, CTA, and compliance risks. Do not invent property features or alter material facts.
```

## Production checklist

- Compare output frame-by-frame against real photos.
- Remove clips that create fake views, extra rooms, different finishes, or misleading scale.
- Add captions and CTA in editor, not inside model when possible.
- Export compressed versions for social and landing pages.
- Track links with UTMs by platform.

## CRM and KPIs

Tag AI-assisted video links in the CRM as `ai-video`, `listing:[slug]`, `channel:[reel/ad/youtube]`, and `asset:[hook]` so leads can be attributed back to the creative.

- 3-second hold and average watch time
- Profile clicks and property page visits
- Guide downloads/showing requests from video UTMs
- Retargeting audience growth
- Seller satisfaction and no compliance revisions
