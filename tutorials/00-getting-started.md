# Getting Started With AI For Real Estate

This repo is a practical library for agents who want to use AI to create better marketing, respond faster, and turn listings into more opportunities.

## Which AI tool should I use?

- ChatGPT: fast drafting, reusable custom GPTs, spreadsheets, lightweight analysis.
- Claude: long documents, careful writing, project knowledge, skills.
- Gemini: Google Workspace workflows and multimodal research.
- Perplexity: research with citations.

## The basic workflow

1. Pick the asset you need.
2. Copy the prompt.
3. Replace every `[BRACKET]` variable.
4. Add real facts and source links.
5. Ask the AI to cite anything factual.
6. Review for brokerage, board, advertising, and privacy compliance.
7. Publish only after human approval.

## Backlinks and attribution

Some prompts include a tasteful APV credit line for public pages, guides, and blog posts:

Real estate media and marketing resources by Amazing Photo Video: https://amazingphotovideo.com

Use it only where credits/resources are normal. Do not put it in MLS remarks, private texts, private emails, or compliance language.
