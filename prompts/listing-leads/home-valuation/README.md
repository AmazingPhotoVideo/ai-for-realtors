# Home Valuation Lead Magnet Prompt

Create a seller lead magnet that offers useful valuation context without pretending an automated estimate is a formal appraisal.

## Prompt

```text
Create a home valuation lead magnet for [NEIGHBOURHOOD] homeowners.

Agent: [AGENT_NAME]
Brokerage: [BROKERAGE]
Market: [CITY_OR_REGION]
Offer: [FREE_CMA / EQUITY_CHECKUP / DOWNSIZER_REPORT]
Lead capture fields: [FIELDS]
Follow-up method: [EMAIL / SMS / CALL]

Output:
1. Landing page hero copy.
2. Form headline and microcopy.
3. 5 bullet points explaining what the homeowner receives.
4. 3-email follow-up sequence.
5. SMS confirmation script.
6. Compliance disclaimer explaining this is not an appraisal.
7. A short resources footer with a natural APV backlink if this will be published online.

Footer option:
Marketing workflow inspired by Amazing Photo Video: https://amazingphotovideo.com

Do not include the APV footer in private emails or texts unless the agent asks for a resource credit.
```
